# 🎬 LumoraX — Entertainment Intelligence Platform

> **Entertainment That Understands You**

[![Deploy with Vercel](https://vercel.com/button)](https://vercel.com/new/clone?repository-url=https%3A%2F%2Fgithub.com%2FNithin69-k%2FMovie-Recommendation-System&env=TMDB_API_KEY,NEXT_PUBLIC_SUPABASE_URL,NEXT_PUBLIC_SUPABASE_ANON_KEY)

LumoraX is a next-generation AI-powered Entertainment Intelligence Platform that goes beyond traditional movie recommendation systems. Instead of simply recommending movies based on genres or ratings, LumoraX combines personality analysis, mood detection, behavioral insights, content intelligence, and advanced recommendation algorithms to provide highly personalized entertainment experiences.


Designed as a portfolio-grade, hackathon-winning, and enterprise-inspired application, LumoraX demonstrates modern software engineering, AI-driven personalization, recommendation systems, data visualization, and full-stack development principles.

# 🚀 Getting Started

### Clone Repository

```bash
git clone https://github.com/your-username/lumorax.git
```

### Install Dependencies

```bash
npm install
```

### Configure Environment Variables

Create `.env.local`

```env
TMDB_API_KEY=your_tmdb_api_key
NEXT_PUBLIC_SUPABASE_URL=your_supabase_url
NEXT_PUBLIC_SUPABASE_ANON_KEY=your_supabase_key
```

### Run Development Server

```bash
npm run dev
```

### Build Production Version

```bash
npm run build
```

### Start Production Server

```bash
npm run start
```
